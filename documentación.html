Objetivo General del Proyecto

Desarrollar un sitio web personal utilizando exclusivamente HTML5 y CSS para practicar la estructuración semántica de contenido, diseño básico de interfaces y organización de páginas interconectadas. Este proyecto busca demostrar la capacidad de integrar múltiples páginas web con navegación funcional, empleando estilos CSS en línea e internos para darle coherencia visual al sitio.

Tecnologías Utilizadas

  • HTML5: Para estructurar todo el contenido del sitio de manera semántica.
  • CSS3: Aplicado dentro de la etiqueta <style> en cada archivo HTML.
  • Enlaces internos: Para navegación entre las páginas del sitio.
  • Enlaces externos: Usados en las secciones de Fotos y Recomendaciones.

Estructura del Sitio Web

  • Index.html: Página principal con navegación, presentación personal, misión y visión.
  • Infancia.html: Biografía de la niñez con diseño centrado y fondo animado.
  • Adolescencia.html: Experiencias educativas, laborales y personales de la adolescencia.
  • Personal.html: Metas, pasatiempos, gustos, disgustos y personajes admirados.
  • Fotos.html: Galería de imágenes representativas con diseño en grid.
  • Recomendaciones.html: Tarjetas con enlaces útiles a sitios web educativos.

Funcionalidades Clave

  • Navegación entre páginas mediante enlaces relativos.
  • Botón flotante para volver al inicio.
  • Diseño coherente y moderno en todas las páginas.
  • Uso semántico de HTML y buenas prácticas de CSS.

Conclusión General

Este proyecto sirvió como una excelente oportunidad para aplicar conocimientos básicos de HTML y CSS. Permitió estructurar contenido de forma semántica, crear una interfaz amigable y práctica, e integrar navegación entre páginas. Fue desarrollado sin frameworks ni JavaScript, demostrando uso eficiente del HTML puro y CSS interno.
